# HTML

Las etiquetas HTML son elementos utilizados para estructurar y dar formato a contenido en un documento HTML.

<"html">: Define el comienzo de un documento HTML.

<"head">: Contiene información meta y enlaces a recursos externos.

<"title">: Define el título de la página web que se muestra en la pestaña del navegador.

<"body">: Contiene el contenido visible de la página web.

<"header">: Se utiliza para definir una sección de encabezado.

<"footer">: Se utiliza para definir una sección de pie de página.

<"h1">, <"h2">, <"h3">, <"h4">, <"h5">, <"h6">: Encabezados de diferentes niveles.

<"p">: Define un párrafo de texto.

<"a">: Crea un hipervínculo o enlace.

<"img">: Inserta una imagen en la página web.

<"ul">: Define una lista no ordenada.

<"ol">: Define una lista ordenada.

<"li">: Elemento de lista dentro de <"ul"> o <"ol">.

<"div">: Se utiliza para agrupar y contener elementos HTML para aplicar estilos y manipulaciones.

<"span">: Se utiliza para aplicar estilos o manipular texto en línea.

<"table">: Define una tabla.

<"tr">: Define una fila en una tabla.

<"td">: Define una celda de datos en una tabla.

<"th">: Define una celda de encabezado en una tabla.

<"form">: Define un formulario para recopilar datos del usuario.

<"input">: Define un campo de entrada en un formulario.

<"button">: Define un botón en un formulario.
